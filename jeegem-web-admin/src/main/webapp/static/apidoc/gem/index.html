GEM方法主要分类：

序号分类用途
1Object封装对象方法
2Dict封装字典方法
3Page封装分页方法
4Model封装弹出框方法
5Validate封装验证方法
6Date封装日期方法
7Tags封装特殊标签方法

1.GEM.Object

GEM.Object.notNull(obj)
断某对象不为空..返回true 否则 false
GEM.Object.notEmpty(obj)
判断某对象不为空..返回obj 否则 ""
返回顶部

2.GEM.Dict

/*
 * ids   对应id值(多个逗号分隔)
 * keys  对应key值(多个逗号分隔)
 * type  (可选)1.请选择，2.自定义数组。默认不填
 * dfstr (可选)自定义数组
 */
GEM.Dict.setSelect(ids,keys,type,dfstr)
根据字典去数据库寻找值填充select选择框
返回顶部

3.GEM.Page

/*
 *设置分页方法
 *formId 分页参数Form的Id
 *pageId 分页位置Id
 *pagesize 分页显示数量
 *pagenum 页码
 *totalCount 数据总数
 *fun 获得数据方法名
*/
GEM.Page.setPage(formId,pageId,pagesize,pagenum,totalCount,fun)
/*
 *跳转分页
 *formId 分页参数Form的Id
 *num 页码
 *JpFun 跳转方法
 */
GEM.Page.jump(formId,pagenum,JpFun)
/*
 *设置分页单个显示数量
 *formId 分页参数Form的Id
 *size 显示数量
 *JpFun 跳转方法
 */
GEM.Page.setSize(formId,size,JpFun)
/*
 *自定义跳转分页
 *formId 分页参数Form的Id
 *pageId 分页Id值
 *JpFun 跳转方法
*/
GEM.Page.jumpCustom(formId,pageId,leng,JpFun)
返回顶部

4.GEM.Model

/*
 *loading框
 *Str loading的语句
 */
GEM.Model.loading(Str)
/*
 *关闭loading框
 *Str 
 */
GEM.Model.loadingClose()
/*
 *确认框
 *fun 确认调用后执行方法名
 *Str 询问语句可以是html格式
 */
GEM.Model.confirm(fun,Str)
/*
 *提示信息框
 *Str 语句可以是html格式（如：<span>保存成功</span>）
 *fun 方法名可带参数 例如：a('xx') 
 */
GEM.Model.info(Str,fun)
/*
 *提示错误框
 *Str 语句可以是html格式（如：<span>保存成功</span>）
 */
GEM.Model.error(Str)
返回顶部

5.GEM.Validate

/*
 *判断是否为空有提示和返回值
 *是空返回true，不是空返回false
 *id
 *side 提示窗显示位置  1，2，3，4 分别代表 上右下左 默认为1（上） 可选
 */
GEM.Validate.isnull(id,side)(新版取消)
/*
 *表单验证
 *fromId
 *使用方法 在表单必须使用gemValidate属性
 *required :必要字段
 *date     :日期
 *datetime :日期+时间
 */
GEM.Validate.form=function(fromId,side)
/*
 *判断是否是电子邮箱
 *是返回true，不是返回false
 */
GEM.Validate.isEmail(email)
/*
 *判断是否是日期
 *是返回true，不是返回false
 */
GEM.Validate.isDate(date)
/*
 *判断是否是日期时间
 *是返回true，不是返回false
 */
GEM.Validate.isDatetime(datetime)
返回顶部

6.GEM.Date

/*
 *时间格式化(默认)
 *time 时间
 */
GEM.Date.Default(time)
/*
 *时间格式化
 *time 时间
 *fmt  格式
*/
GEM.Date.Format(time,fmt)
返回顶部

7.GEM.Tags

/*
 *设置选择框1或0值
 *class是isValidCheckbox的选择框Yes或No，value设为1或0
 *formId form的Id
 */
GEM.Tags.isValid(formId)
返回顶部
